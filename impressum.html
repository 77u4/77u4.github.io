Theme

Diese Website wurde auf Geschwindigkeit, Barrierefreiheit, SEO-Freundlichkeit und für Mobilgeräte optimiert. Über die Jahre ist so ist aus diesem kleinen, nebenbei laufenden Projekt ein schlichtes und leicht anzupassendes Theme geworden, das gerne kopiert und weiterentwickelt werden kann.

Den Quellcode habe ich unter MIT-Lizenz auf GitHub verfügbar gemacht.
